観測係・テラちゃん

地球のいちばん近くで「ちょうどよさ」を記録する係

テラちゃんはじめまして!地球の観測係、テラちゃんだよ

プロフィール

SHUZOKU / 正体
地球を見守ってきた観測衛星たちの「想い」が集まって生まれた、小さないきもの。だから体にはアンテナとソーラーパネルの名残がある。観測員としては見習いで、担当はいきなり地球という大役。
RELATION / 地球との関係
地球そのものではない(地球の擬人化ではない)。地球のまわりを回りながら見つめる「観測する側」の住人で、体の色も地球の青ではなく観測コンソールと同じテラル色。観測対象であり、いちばんの友だち。
VOICE / 口調
フランクで素直。むずかしい言葉より先に感想が出る(「暑い…暑すぎるよ…」)。パニックにはならず、素朴なリアクション専門。一人称は「ボク」。
LIKE / 好きなもの
夕焼け、オーロラ、平均気温15℃、24時間ぴったりの一日。
DISLIKE / 苦手なもの
金星の話(暑いから)、急に速くなる自転(目が回るから)。
HIMITSU / ひみつ
頭のアンテナ灯は、観測データを受信するとチカチカ光る。地球の気温や自転のようすを、さわらなくても感じ取れる。パネルの羽は飾りじゃなくて、太陽の光でおなかを満たすためのもの。
FUWAFUWA / 浮いている理由
からだがとてもとても軽くて、地球の重力にほとんどつかまらないから、いつもふわふわ。重い星ではちょっと沈むし、軽い星ではふわんと高く浮いてしまう(統合シミュレーターの重力スライダーで確かめられるよ)。
TAIL / データの尾
機体の下から流れるオーロラ色の光は、観測データが流れているところ。きれいなものを見つけると、ちょっとだけ長くなる…らしい。
KUCHIGUSE / 口ぐせ
「記録しまーす!」「観測ログに書いとくね」。うれしいときは語尾がのびる(「ちょうどいい〜!」)。
YUME / ゆめ
一人前の観測員になって、いつか自分の力で「地球みたいにちょうどいい星」を見つけること。地球は教科書で、お手本で、いちばんの友だち。

表情ログ

COLD / 寒い
DIZZY / 目が回る
HAPPY / ちょうどいい
